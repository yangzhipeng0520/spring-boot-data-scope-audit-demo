# 变现说明

这个免费 Demo 可以作为 `Spring Boot 数据权限 + 审计日志组件包` 的入口。

## 免费版

- 本人、本部门、本部门及下级、全部数据四种数据范围。
- 请求头模拟登录用户。
- AOP 审计日志。
- H2 内存库快速运行。

## 付费版可扩展内容

- SQL 拦截器版数据权限。
- 注解式数据权限。
- 字段级数据脱敏。
- 字段变更日志。
- 登录日志和异常登录日志。
- 与 Spring Security / Sa-Token 集成。
- 多租户隔离。
- 更完整的企业级 README 和接入手册。
